À propos de moi



  • Lauryne LAGE
  • 26 ans
  • En formation de développeur web

Je suis un développeur web passionné, avec une expérience solide dans la conception et le développement de sites web élégants, intuitifs et fonctionnels. J'aime relever des défis techniques et suis constamment à la recherche de nouvelles technologies et approches innovantes pour améliorer mes compétences. Ma passion pour le développement web me pousse à créer des solutions efficaces, avec un fort accent sur l'expérience utilisateur.

Depuis décembre 2022, j'ai eu l'opportunité de travailler en alternance chez QuickInfoService, une expérience déterminante qui a marqué mon parcours professionnel. Pendant ces deux années d'alternance, j'ai acquis une expertise pratique en développement web tout en perfectionnant mes connaissances en HTML, CSS, JavaScript, ainsi que sur des frameworks modernes comme React et Vue.js.

Chez QuickInfoService, j'ai travaillé sur une grande variété de projets, allant de la refonte complète de sites web à la création de plateformes interactives pour les clients. J'ai eu l'opportunité de travailler sur plusieurs plateformes de création de sites, telles que PrestaShop et WooCommerce, renforçant ainsi mes compétences en e-commerce.

Durant ma formation, j'ai également découvert le métier de Community Manager, ce qui m'a permis de travailler avec les réseaux sociaux, d'effectuer des montages vidéo, des retouches photo, ainsi que de créer des contenus textuels engageants. De plus, j'ai acquis une bonne expérience dans l'optimisation des performances des sites web, réduisant ainsi les temps de chargement et améliorant le référencement naturel (SEO).

Cette expérience chez QuickInfoService, d'une durée de presque deux ans, a non seulement consolidé mes compétences techniques, mais elle m'a aussi permis de développer une solide capacité à gérer des projets de bout en bout, que ce soit de manière autonome ou en équipe. Mon esprit d'analyse, combiné à mon goût pour les défis techniques, a renforcé ma volonté d'explorer et de créer des solutions numériques toujours plus innovantes.

Mes Projets

Projet 1 - Créez la page d'accueil d'une agence de voyage avec HTML & CSS

Ce projet m'a appris à créer une page d'accueil responsive pour une agence de voyage en utilisant HTML et CSS, tout en intégrant des maquettes Figma pour différents appareils.


  • Implémenter une interface responsive avec HTML et CSS
  • Installer un environnement de développement front-end
  • Intégrer du contenu conformément à une maquette avec HTML et CSS

Projet 2 - Créez une page web dynamique avec JavaScript

Ce projet m'a fait créer une page web dynamique pour le site internet d'une architecte d'intérieur en utilisant JavaScript et en communiquant avec une API.


  • Gérer les événements utilisateurs avec JavaScript
  • Manipuler les éléments du DOM avec JavaScript
  • Récupérer les données utilisateurs dans le JavaScript via des formulaires

Projet 3 - Optimisez le référencement d'un site de photographe

Ce projet m'a demandé d'identifier les problèmes de chargement et de référencement du site web d'un photographe, et de les corriger pour améliorer les performances globales.


  • Optimisation complète d'un site web
  • Amélioration des performances de chargement
  • Optimisation du référencement naturel (SEO)

Projet 4 - Implémentez le front-end d'une application bancaire avec React

Ce projet m'a fait travailler sur le développement front-end d’une application bancaire en utilisant React et Redux pour créer une expérience utilisateur dynamique et réactive.


  • Afficher les données du back end sur l'interface via des appels API
  • Configurer des routes API pour la communication client / serveur
  • Implémenter la gestion des données avec Redux pour assurer le fonctionnement du front
